## 教程 1 概述

这篇教程覆盖 4 个具体的研究场景，每个场景都按 **"你在做什么方向的研究 → 手头缺什么 → AIG 里有现成的什么 → 如何提取与改造 → 你得到了什么 → 对论文的贡献"** 展开：

| 场景 | 研究方向 | AIG 提供的数据 | 论文贡献 |
|------|---------|---------------|---------|
| 一 | AI 基础设施漏洞统计实证研究 | 88 组件 1252 条 CVE | Dataset Overview + RQ1-3 + 趋势分析 |
| 二 | CVSS 度量与攻击面特征分析 | 1138 条 CVSS 3.x 向量 | 攻击门槛分析 + CIA 热力图 + 威胁模型 |
| 三 | LLM 越狱攻击与安全对齐评测 | 16 个数据集 7000+ prompts | ASR 矩阵 + 中英文交叉评测 + 分类细粒度分析 |
| 四 | AI 安全研究空白与攻击面测绘 | 121 指纹 vs 88 漏洞组件 | SoK 组件分类学 + 覆盖率 + Research Agenda |
